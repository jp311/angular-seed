使用 [RequireJS](http://requirejs.cn/) 进行依赖管理。

## 目录结构

- **组件包名称**
 - **/controllers**：包含控制器代码。
   - **controllers.js**：控制器配置、定义模块。
 - **/directives**：包含指令代码。
   - **directives.js**：指令定义模块。
 - **/filters**：包含过滤器代码。
   - **filters.js**：过滤器定义模块。
 - **/services**：包含服务代码。
   - **services.js**：服务定义模块。
 - **/views**：包含视图文件，HTML 代码片段。
 - **main.js** 组件包定义模块。
